# Install

```
poetry install 
```

# Usage

```python
usage: resizeall.py [-h] [--ignore-ext] path

resize all images files found in a given folder

positional arguments:
  path          image file path or folder path containing images to resize

optional arguments:
  -h, --help    show this help message and exit
  --ignore-ext  ignore file extensions, in this case, try to open all files and ignore if an exception occurs (phpbb file names does not include extensions)
```
